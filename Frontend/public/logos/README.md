# Logos del proyecto

Coloca aquí los archivos de logo (WebP, SVG, PNG, etc.) que quieras usar con **URL fija**.

- **Ruta en la app:** `/logos/nombre-del-archivo.webp` (recomendado WebP para mejor rendimiento)
- **Ejemplo en HTML/componente:** `src="/logos/IMGReservaYa.webp"`
- **Ejemplo en CSS:** `url('/logos/logo-mesa.svg')`

Estos archivos no pasan por Vite; se sirven tal cual desde la raíz.
